# نادي إبراهيم للقراءة

موقع إلكتروني لنادي القراءة العربي

## الملفات

- `index.html` - الصفحة الرئيسية للموقع
- `styles.css` - ملف التنسيقات والتصميم
- `script.js` - ملف JavaScript للتفاعلات
- `favicon.svg` - أيقونة الموقع
- `manifest.json` - ملف الويب مانيفست
- `.nojekyll` - ملف لتعطيل معالجة Jekyll على GitHub Pages

## خطوات النشر على GitHub Pages

### الخطوة 1: إعداد المستودع

1. أنشئ مستودعًا جديدًا على GitHub أو استخدم مستودعًا موجودًا
2. تأكد من أن جميع الملفات موجودة في المستودع:
   ```
   index.html
   styles.css
   script.js
   favicon.svg
   manifest.json
   .nojekyll
   README.md
   ```

### الخطوة 2: تفعيل GitHub Pages

1. اذهب إلى مستودع GitHub
2. اضغط على **Settings** (الإعدادات) في أعلى الصفحة
3. في القائمة الجانبية، اضغط على **Pages** (الصفحات)
4. تحت **Source** (المصدر)، اختر:
   - **Branch: main** (أو master)
   - **Folder: / (root)**
5. اضغط على **Save** (حفظ)

### الخطوة 3: الانتظار للنشر

- بعد حفظ الإعدادات، سيبدأ GitHub في نشر الموقع
- قد يستغرق الأمر بضع دقائق
- ستجد رابط الموقع في نفس صفحة الإعدادات بعد النشر
- الرابط سيكون على الشكل: `https://[username].github.io/[repository-name]`

### الخطوة 4: التحقق من النشر

1. افتح الرابط في المتصفح
2. تأكد من ظهور الموقع بشكل صحيح
3. تحقق من أن أيقونة الموقع (favicon) تظهر في تبويب المتصفح

## تحديث الموقع

لإجراء تغييرات على الموقع:

1. عدّل الملفات محليًا
2. ارفع التغييرات إلى GitHub:
   ```bash
   git add .
   git commit -m "تحديث الموقع"
   git push
   ```
3. سيتم تحديث الموقع تلقائيًا خلال دقائق

## التخصيص

### تعديل النصوص
- عدّل النصوص في `index.html`

### تغيير الألوان
- عدّل المتغيرات في بداية ملف `styles.css`:
  ```css
  :root {
    --primary-color: #3d2817;
    --secondary-color: #8b6f47;
    ...
  }
  ```

### إضافة كتب جديدة
- أضف بطاقات كتب جديدة في قسم "الكتب التي تمت قراءتها سابقاً" في `index.html`

### تغيير Typeform
- عدّل `data-tf-live` في قسم الاشتراك في `index.html`
